## 修改密码

1. 登录 [Momoapi 控制台](https://momoapi.top/dashboard/overview)
2. 进入「个人设置」或「账户设置」
3. 找到「修改密码」选项
4. 输入当前密码和新密码
5. 确认保存

<Warning>
  修改密码后，所有已创建的 Token 仍然有效。如果需要让所有设备重新登录，请在 Token 管理页面删除所有旧 Token。
</Warning>

## 忘记密码

如果你忘记了密码：

1. 在登录页面点击「忘记密码」
2. 输入注册时使用的邮箱
3. 查收重置密码的邮件
4. 按邮件中的链接设置新密码
