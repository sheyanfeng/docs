# 注册账号

> 创建你的 Momoapi 平台账号

## 注册流程

1. 访问 [Momoapi 控制台](https://momoapi.top/dashboard/overview)
2. 点击「注册」或「Sign Up」
3. 填写注册信息：
   - **邮箱**：用于登录和接收通知
   - **用户名**：你的显示名称
   - **密码**：设置一个安全密码
4. 点击「注册」完成

## 注册后

注册成功后，你可以：

- 创建 [API Token](/getting-token) 用于客户端配置
- 查看 [可用模型与定价](/models/overview)
- 配置 [Claude Code / Codex](/cli-config/env-check) 客户端
