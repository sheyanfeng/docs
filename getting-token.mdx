# 获取 API Token

> 了解如何创建和管理你的 API Token

Token 是你使用 Momoapi API 服务的凭证。所有客户端都需要配置 Token 才能正常工作。

## 什么是 Token

Token 是一串以 `sk-` 开头的字符串，例如 `sk-abc123def456`。它用于验证你的身份和计费。

## 关键信息

在使用 Token 配置客户端时，你需要以下信息：

| 参数 | 值 |
| --- | --- |
| **Base URL** | `https://momoapi.top` |
| **API Key** | 你创建的 Token（`sk-xxx`） |

> **重要**：Base URL 和 API Key 缺一不可，两者都需要正确配置。

## 创建 Token

1. 访问 [Momoapi 控制台](https://momoapi.top/dashboard/overview)
2. 使用你的账号登录
3. 进入「令牌」或「Token」管理页面
4. 点击「创建新 Token」
5. 设置名称（方便你识别用途），确认创建
6. 复制生成的 Token（以 `sk-` 开头）

<Warning>
  Token 创建后只显示一次，请务必立即复制保存。如果丢失，需要删除后重新创建。
</Warning>

## Token 安全

- **不要分享**：Token 等同于你的账户密码，不要发送给他人
- **不要提交到 Git**：不要将 Token 写入代码仓库
- **定期轮换**：建议定期删除旧 Token，创建新 Token
