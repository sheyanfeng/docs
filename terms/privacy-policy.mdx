生效日期：2026年3月3日

### 1. 简介

本隐私政策描述了 Momo API（"我们"）在您使用我们的 AI API 聚合服务时如何收集、使用和保护您的个人信息。

### 2. 我们收集的信息

- **账户信息**：电子邮件地址、用户名和密码、支付和账单信息。
- **使用数据**：API 调用日志（使用的模型、Token 数量、时间戳）、IP 地址和设备信息。
- **内容数据**：通过 API 发送的输入（提示词）、AI 模型生成的输出（响应）。

### 3. 我们如何使用您的信息

- 提供和维护服务。
- 处理付款和管理您的账户。
- 监控使用情况以进行计费和速率限制。
- 提高服务质量和安全性。
- 就服务更新与您沟通。
- 遵守法律义务。

### 4. 数据存储与安全

- 您的数据存储在安全的服务器上。
- 我们采用行业标准的安全措施来保护您的信息。
- API 密钥在静态时加密存储。
- 我们出于计费和合规目的保留使用日志。

### 5. 数据共享

我们不会：

- 将您的个人信息出售给第三方。
- 使用您的数据训练 AI 模型。
- 与第三方共享您的数据，除非为提供服务所必需（例如，将 API 请求转发给上游 AI 提供商）。

### 6. 数据保留

- 账户信息在您的账户处于活跃状态时保留。
- 使用日志出于计费目的保留最多 90 天。
- 内容数据（输入/输出）不会永久存储，仅作瞬时处理。
- 您可以随时请求删除您的账户和相关数据。

### 7. 您的权利

- 访问您的个人数据。
- 更正不准确的数据。
- 请求删除您的数据。
- 导出您的数据。
- 随时撤回同意。

### 8. Cookie

我们使用必要的 Cookie 用于会话管理和身份验证。不使用第三方跟踪 Cookie。

### 9. 儿童隐私

本服务不面向 18 岁以下的个人。我们不会故意收集儿童的信息。

### 10. 政策变更

我们可能不时更新本隐私政策。我们将通过电子邮件或服务通知您重大变更。

### 11. 联系我们

如对本隐私政策有疑问或关切，请联系：[support@momoapi.top](mailto:support@momoapi.top)
