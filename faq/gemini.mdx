### Gemini CLI 连接失败

通过 Momo API 使用 Gemini CLI 时，先确认环境变量已经设置：

```bash
echo $GEMINI_API_KEY
# 应显示你的令牌
```

如果变量为空，请重新配置令牌并打开新的终端窗口。

还需要确认当前 API Key 的模型分组包含 Gemini 模型；如果分组不包含 Gemini，CLI 会连接失败或提示模型不可用。

### 图像生成失败

使用 Nano Banana Pro 生成图像失败时，按顺序检查：

1. 模型名称是否填写为 `gemini-3-pro-image-preview`。
2. 提示词是否可能触发使用政策限制。
3. 当前账户额度是否充足。
4. 当前令牌是否有图像生成模型权限。

### 中文回复不稳定

Gemini 模型在某些场景下中文回复稳定性可能不如英文。可以尝试：

- 使用更明确的中文指令，例如“请全程使用中文回答”。
- 对复杂任务先用英文提示词描述目标，再要求输出中文结果。
- 如果需要稳定中文长文本，可切换到更适合中文写作或分析的模型。
